# Panduan Penggunaan Kategori (Categories)

Modul **Categories** membantu Anda mengelompokkan barang dagangan agar:

1.  Mudah dicari di mesin kasir (POS).
2.  Laporan penjualan lebih rapi (bisa lihat "Penjualan Minuman" vs "Penjualan Makanan").

## 1. Membuat Kategori

1.  Buka menu admin, cari bagian **Manage Categories**.
2.  Tekan tombol **+ New Category**.
3.  Isi nama kategori (misal: "Minuman Dingin", "Sembako", "Rokok").
4.  Tekan **Create**.

## 2. Mengatur Produk

Setelah kategori dibuat:

1.  Pergi ke menu **Products**.
2.  Edit produk yang ada.
3.  Pilih **Category** yang baru dibuat tadi.

## Tips

- Gunakan nama yang pendek dan jelas agar tombol di mesin kasir (POS) mudah dibaca.
- Kategori yang tidak aktif/dihapus tidak akan muncul di POS, tapi produk di dalamnya tidak ikut terhapus (hanya jadi tanpa kategori).
