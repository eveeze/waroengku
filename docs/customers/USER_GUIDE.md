# Panduan Pelanggan & Kasbon (Customers)

Modul ini mengelola data pelanggan dan hutang-piutang (Kasbon).

## 1. Menambah Pelanggan

1.  Buka menu **Customers**.
2.  Tekan **+ New Customer**.
3.  Wajib: **Nama**.
4.  Opsional: **No HP** (agar bisa dikirimi tagihan via WA).
5.  **Credit Limit**: Batas maksimal hutang. Misal Rp 500.000. Jika hutang sudah segini, dia tidak bisa kasbon lagi.

## 2. Transaksi Kasbon (Belanja Dulu Bayar Nanti)

1.  Saat Checkout di POS, pilih pelanggan.
2.  Pilih metode bayar **Kasbon**.
3.  Transaksi selesai, hutang pelanggan otomatis bertambah.

## 3. Bayar Hutang (Pelunasan)

Saat pelanggan datang untuk bayar hutang:

1.  Buka menu **Customers**.
2.  Cari nama pelanggan. Indikator merah berarti punya hutang.
3.  Buka detailnya, tekan **Pay Debt / Bayar Hutang**.
4.  Masukkan jumlah yang dibayar (bisa nyicil/sebagian).
5.  Hutang otomatis berkurang.

## Tips

- Rutin cek menu **Customers > With Debt** untuk melihat siapa saja yang belum lunas.
- Anda bisa menagih via WhatsApp menggunakan nomor yang tersimpan.
