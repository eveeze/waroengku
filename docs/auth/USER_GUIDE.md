# Panduan Login & Akun (Auth)

## 1. Login (Masuk Aplikasi)

1.  Buka aplikasi Warungku.
2.  Masukkan **Email** dan **Password** yang diberikan oleh Admin.
3.  Tekan **Login**.
4.  Jika berhasil, Anda akan diarahkan ke halaman utama sesuai peran (Role) Anda.

## 2. Peran Pengguna (Roles)

Setiap akun memiliki akses berbeda:

- **Admin**: Bisa akses SEMUA menu (Produk, Laporan, User, dll).
- **Cashier**: Fokus ke Transaksi (POS), Kasbon, dan Cash Flow. Tidak bisa edit stok/hapus produk.
- **Inventory**: Fokus ke Stok, Kulakan, dan Stock Opname.

## Tips Keamanan

- Jangan berikan password Anda ke orang lain.
- Jika Anda staf baru, minta Admin membuatkan akun baru, jangan pakai akun orang lain.
- Selalu **Logout** jika shift Anda sudah selesai untuk mencegah penyalahgunaan.
