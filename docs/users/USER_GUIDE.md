# Panduan Manajemen Staf (Users)

Hanya Admin yang bisa mengakses menu ini.

## 1. Menambah Karyawan Baru

1.  Buka menu **More > Users**.
2.  Tekan **+ Add User**.
3.  Isi Nama, Email, dan Password sementara.
4.  Pilih Role (Peran):
    - **Cashier**: Untuk kasir.
    - **Inventory**: Untuk orang gudang.
    - **Admin**: Untuk pemilik/manajer.

## 2. Mengelola Akun

- **Reset Password**: Jika staf lupa password, edit user tersebut dan masukkan password baru.
- **Nonaktifkan Akun**: Jika staf berhenti bekerja, edit statusnya menjadi **Inactive** (jangan dihapus agar riwayat transaksinya tetap ada).
