# Panduan Pembayaran (Payments)

Sistem Warungku mendukung berbagai jenis pembayaran untuk memudahkan pelanggan.

## 1. Tunai (Cash)

Metode paling umum.

- Pastikan kasir memasukkan jumlah uang yang diterima dengan benar agar hitungan kembalian akurat.

## 2. QRIS (Midtrans)

Pembayaran digital otomatis.

1.  Saat checkout, pilih **QRIS**.
2.  Sistem akan menampilkan kode QR di layar (atau di tablet kasir).
3.  Minta pelanggan scan pakai GoPay/OVO/ShopeePay/BCA Mobile.
4.  Tunggu notifikasi "Pembayaran Berhasil" di layar kasir. (Tidak perlu cek manual).

## 3. Transfer Bank

Untuk pembayaran manual ke rekening toko.

1.  Kasir mengecek mutasi rekening (via Mobile Banking toko).
2.  Jika uang masuk, pilih metode **Transfer** di aplikasi.
3.  Opsional: Tulis catatan ("Transfer BNI a.n Budi").

## 4. Kasbon (Hutang)

Lihat panduan lengkap di modul **Customers**. Pelanggan bisa bayar nanti.
