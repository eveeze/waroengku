# Panduan Laporan (Reports)

Modul ini khusus Admin untuk memantau performa warung.

## 1. Dashboard Utama

Saat login sebagai Admin, Anda langsung melihat ringkasan:

- **Today's Sales**: Omzet hari ini.
- **Low Stock**: Barang yang mau habis.
- **Total Kasbon**: Uang yang masih di tangan pelanggan.

## 2. Laporan Harian (Daily Report)

1.  Buka menu **Reports**.
2.  Pilih tanggal.
3.  Lihat detail:
    - Total Transaksi.
    - Total Omzet.
    - Estimasi Keuntungan (Laba Kotor).

## 3. Analisa

- **Top Products**: Barang apa yang paling laku?
- **Kasbon**: Siapa yang hutangnya paling banyak?
  Gunakan data ini untuk keputusan kulakan (beli yang laku saja) atau penagihan hutang.
