# Panduan Titip Jual (Consignment)

Modul ini untuk mengelola barang titipan orang lain (Konsinyasi).

## 1. Menambah Penitip (Consignor)

1.  Masuk menu **Consignment**.
2.  Tambah data orang yang menitip barang (Supplier).
3.  Catat nama dan info kontaknya.

## 2. Menghubungkan Produk

Agar sistem tahu produk A milik Si B:

1.  Buka menu **Products**.
2.  Edit/Buat Produk.
3.  Cari kolom **Consignor / Supplier**.
4.  Pilih nama penitipnya.

## 3. Cek Penjualan

Untuk tahu berapa uang yang harus disetor ke penitip:

1.  Buka menu **Consignment**.
2.  Pilih nama penitip.
3.  Sistem akan menampilkan laporan: "Produk X laku 10 pcs, Total Rp 50.000".
