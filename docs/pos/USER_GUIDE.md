# Panduan Mesin Kasir (POS)

**POS (Point of Sale)** adalah halaman utama yang digunakan kasir sehari-hari untuk melayani pembeli.

## 1. Menahan Transaksi (Hold Cart)

Gunakan ini jika pelanggan ketinggalan dompet atau ingin menambah barang, dan ada antrian di belakangnya.

1.  Masukkan barang pelanggan A ke keranjang.
2.  Tekan tombol **Hold Cart** (ikon Jam/Pause).
3.  Keranjang akan kosong, Anda bisa melayani Pelanggan B.
4.  Untuk memanggil kembali belanjaan A, tekan tombol **Held Carts** di samping.
5.  Pilih nama pelanggan/waktu hold.
6.  Barang akan kembali ke keranjang.

## 2. Refund (Pengembalian Barang)

Jika pelanggan ingin mengembalikan barang yang sudah dibeli:

1.  Buka menu **Transactions / History**.
2.  Cari transaksi yang dimaksud.
3.  Tekan **Refund**.
4.  Pilih barang yang dikembalikan.
5.  PENTING: Centang **"Return to Inventory"** jika barangnya masih bagus dan bisa dijual lagi. Jangan centang jika barang rusak/basi.

## Tips POS

- Gunakan **Barcode Scanner** untuk kecepatan maksimal.
- Jika scanner tidak membaca, coba cari manual dengan mengetik nama produk di kolom pencarian.
