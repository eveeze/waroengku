# Panduan Arus Kas (Cash Flow)

Modul ini mencatat uang fisik di laci kasir (Drawer).

## 1. Buka Kasir (Open Drawer)

Setiap awal shift, Anda wajib melakukan ini:

1.  Hitung uang modal di laci.
2.  Tekan **Open Drawer**.
3.  Masukkan jumlahnya (misal: Rp 200.000).
4.  Ini menjadi "Saldo Awal".

## 2. Catat Pengeluaran (Expense)

Jika Anda mengambil uang dari laci untuk operasional (beli es, pulsa toko, bensin):

1.  Buka menu **Cash Flow**.
2.  Tekan **Add Cash Flow**.
3.  Pilih Tipe: **Expense**.
4.  Pilih Kategori (misal: "Supplies").
5.  Masukkan jumlah dan keterangan.
6.  Tekan **Save**.

## 3. Tutup Kasir (Close Drawer)

Setiap akhir shift:

1.  Hitung semua uang fisik di laci.
2.  Tekan **Close Drawer**.
3.  Masukkan jumlah yang Anda hitung.
4.  Sistem akan memberitahu jika ada **Selisih (Variance)**.
    - **Shortage**: Uang kurang (Negatif).
    - **Surplus**: Uang lebih (Positif).
